# Advanced Salesforce Log Analyzer

A sophisticated Google Chrome extension that enhances Salesforce debug log analysis with advanced features beyond the existing Salesforce Log Inspector.

## Features
- **Raw Log View**: Syntax highlighting, search, filtering.
- **Method Execution Tree**: Collapsible tree structure, execution times.
- **DML Operations Dashboard**: Count of operations, limits.
- **SOQL Query Analyzer**: Identify missing filters, limits, performance.
- **Performance Dashboard**: Visual charts, CPU breakdowns.
- **Governor Limits Tracker**: Real-time visualizations.
- **Error Inspector**: Catch all exceptions.
- **Timeline Visualization**: Interactive event timeline.

## Build Setup
1. Clone the down repository.
2. Go to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" and select the `advanced-salesforce-log-analyzer` directory.
